# Новогодняя Викторина

Интерактивная новогодняя викторина с тематическими вопросами и волшебными эффектами.

## Описание

Это веб-приложение представляет собой интерактивную викторину с новогодней тематикой, включающую:
- 20 различных тематических карточек
- Тематические вопросы для каждой категории
- Анимации и визуальные эффекты
- Специальные режимы и бонусы
- Обратный отсчет до Нового года

## Особенности

- Тематические вопросы для каждой карточки
- Анимации и визуальные эффекты при взаимодействии
- Режим "волшебной скорости"
- Секретные коды и сюрпризы
- Таймер обратного отсчета до Нового года
- Адаптивный дизайн для разных устройств

## Деплой на Vercel

Проект готов к деплою на Vercel с помощью файла `vercel.json`.
